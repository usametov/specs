org.specs.runner

ExamplesTestSuite

class ExamplesTestSuite extends JUnitSuite with ExtendedJUnitSuite

A ExamplesTestSuite is a JUnitSuite reporting the results of a System under test (sus) as a list of examples, represented by ExampleTestCase objects. If an example has subExamples, they won't be shown as sub tests because that would necessitate to run the example during the initialization time.
